### A few handy landmarks:
Open = C6 / Am7 (C E G A C E)
Fret 2 = Dm7 (D F A C)
Fret 5 = Fmaj7 (F A C E) — just slide from fret 2
Fret 7 = Gmaj (G B D) / Bm7 colour (B D F# A)
Fret 12 = Octave repeat of open
